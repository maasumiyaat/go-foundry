# Go Foundry

Welcome to **Go Foundry** â€“ a playground and reference collection for Go programming concepts, algorithms, and utilities. This repository is organized for easy exploration and learning, with each directory focusing on a specific topic or technique in Go.

## Structure Overview

- **main.go**: Entry point for general demos and experiments.
- **go.mod**: Go module definition.
- **array/**: Array utilities and tests.
- **binary_search/**: Binary search implementations and tests.
- **code_snippets/**: Example code snippets and visualizations (e.g., LC2011).
- **crypto/**: Cryptography utilities (AES, HMAC, etc.).
- **deque/**: Double-ended queue implementation and tests.
- **file/**: File handling utilities.
- **functions/**: General-purpose functions.
- **goroutines/**: Concurrency patterns, channels, atomic operations, and more.
- **helloworld/**: Basic "Hello, World!" example and test.
- **integers/**: Integer operations and tests.
- **interfaces/**: Go interface examples.
- **iteration/**: Iteration patterns and tests.
- **logs/**: Logging utilities.
- **panic_recover/**: Panic and recover patterns.
- **pprof/**: Profiling utilities and tests.
- **regex/**: Regular expression utilities.
- **smi/**: Shape interface examples and tests.
- **strings/**: String manipulation utilities and tests.

## Getting Started

1. **Clone the repository:**
   ```zsh
   git clone <repo-url>
   cd go-foundry
   ```
2. **Run examples:**
   ```zsh
   go run main.go
   ```
3. **Run tests:**
   ```zsh
   go test ./...
   ```
